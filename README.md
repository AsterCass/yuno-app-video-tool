# 本地资源互联化

## 说明

使用`frp`作为核心进行进行本地资源/服务的互联化，使得公网可访问

<img src="https://astercasc-web-admin-1256368017.cos.ap-shanghai.myqcloud.com/admin-web-article-img/github_yuno-app-video-tool_1.jpg">

## 已有功能支持

* 整合`FRP`
* 支持本地文件穿透

## 开发中功能

* 支持本地服务穿透
* 整合`FFmpeg`
* 支持`H264`视频文件批量转换
* 支持辅助模块自动下载
* 整合`Shadowsocks`
